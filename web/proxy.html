웹 버전 CORS 오류

현재 웹 브라우저에서 API 서버(http://10.100.54.176:8002)에 직접 접근할 수 없습니다.

이는 CORS (Cross-Origin Resource Sharing) 정책 때문입니다.

해결 방법:

  1. 서버팀 CORS 설정 (권장) 
    app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],  # 또는 특정 도메인
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
  2. 개발용 브라우저 실행 
    chrome.exe --user-data-dir=/tmp/chrome --disable-web-security --disable-features=VizDisplayCompositor
  3. Flutter 웹 앱을 모바일로 테스트
